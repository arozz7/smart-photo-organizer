# Smart Photo Organizer

A local photo organization tool powered by AI for face detection, recognition, and smart tagging.

## Features

- **Local-First**: All photo scanning and processing happens on your device. No cloud uploads.
- **AI Face Detection**: Automatically detects faces in your photo library using TensorFlow.js.
- **Smart Grouping**: Groups similar faces together for easy naming.
- **Photo Library**: Browse your photos with a responsive, virtualized grid view.
- **Privacy Focused**: Your photos stay on your disk.

## Development

This project uses React + TypeScript + Vite + Electron.

### Prerequisites

- Node.js (v18+)

### scripts

- `npm run dev`: Start development server (Vite + Electron)
- `npm run build`: Build for production
- `npm run lint`: Run ESLint analysis
